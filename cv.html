O mě

Přes deset let se věnuji vývoji webových aplikací a informačních systémů. Většinu této doby jsem pracoval jako full stack developer s orientací na PHP a MySQL na straně serveru. Frontend jsem dlouho zpracovával bez frameworků, pouze v HTML, CSS (nebo Sass) a JavaScript.

V PHP jsem naprogramoval vlastní MVC a REST server pro náš rodinný podnik. Také jsem vytvořil JS knihovnu podobnou jQuery, která nabízí velmi podobnou funkcionalitu, ale je podstatně menší.

V posledních letech jsem se specializoval na frontend vývoj ve frameworku React (+ Typescript). Také se zajímám o Vue a Svelte.

Kromě webového vývoje se ve volných chvílích věnuji také Android aplikacím (Java/Kotlin) a destopovým aplikacím (C# + Avalonia). Největším koníčkem je ovšem herní vývoj, kde pracuji s UE5 (C++) a Blenderem.
