O mě

Přes deset let se věnuji vývoji webových aplikací a informačních systémů. Většinu této doby jsem pracoval jako full stack developer s orientací na PHP a SQL na straně serveru. Frontend jsem dlouho zpracovával bez frameworků, pouze v HTML, CSS (nebo Sass) a JavaScript.

V PHP jsem naprogramoval vlastní MVC a REST server pro náš rodinný podnik a také JS knihovnu podobnou jQuery, která ovšem data zpracovává vždy jako pole a je podstatně menší (snadné používání built-in metod pro pole).

Nyní se věnuji přechodu na Frontend. Nadchl mě framework React, proto jsem se rozhodl, že se budu specializovat právě na něj. Kromě Reactu se zajímám o NoSQL řešení aplikací (MongoDB).


Po ukončení studia na UPOLu jsem cítil, že potřebuji změnit prostředí a proto jsem nějakou dobu pracoval v IT pouze na "částečný úvazek". Programování jsem se věnoval po zaměstnání a ve svém volném čase, pracoval jsem především na projektech, které mě bavily a pro naši malou rodinnou firmu.

Nyní se chci opět vrátit k full-time IT profesi - programování webových aplikací. Ideálně frontend, ale rád se naučím i nové technologie a programovací jazyky (nyní se zajímám o Kotlin a Android development).
